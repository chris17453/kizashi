Explorer guide

Useful paths through the evidence.

  • Connector ID and Source type narrow by where a record came from; leave either blank to match all.
  • Search text matches anywhere in the record's raw payload — use it for a quick free-text scan across any source type.
  • Subject, From, and Attachment filename are email-specific fields — they only match records ingested from mail connectors.
  • Combine fields to narrow results — filters are ANDed together, not ORed.
