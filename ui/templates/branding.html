How this works

  • Every field is optional — leaving one blank uses the platform default (product name "Kizashi", the default diamond mark, the default accent color), it never renders broken.
  • The login page loads branding for whichever workspace name is in the "Workspace" field as soon as you tab away from it, so an operator can preview it before signing in.
  • Only Admins can change these — it's a workspace-wide identity change, not a per-user preference.
  • Every save is written to an immutable audit log for compliance review.
