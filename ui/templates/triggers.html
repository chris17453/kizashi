How triggers work

  • "Event type to match" must be a key the AI/ML backend actually returns — see AI Analysis to see or change what it looks for.
  • Fill in the field group matching whichever Condition you selected above — the others are ignored.
  • "Combine multiple sources" fires only once every listed event type has reached its own minimum count for the same entity within the window — e.g. a negative-sentiment email and an unresolved chat message from the same customer, or any other combination of event types (up to 6 sources per trigger).
  • A firing trigger creates an Event, visible on the Events page and traceable back to its source records via Data → record journey.
  • Every create is written to an immutable audit log for compliance review.
